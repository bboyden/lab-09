Lab 9: Release-Based Workflow with GitHub

This lab explored GitHub's release-based workflow by working through a self-directed GitHub Learning Lab course.

Key Skills Learned:

  • Created and explored a GitHub Learning Lab course repo
  • Practiced opening, merging, and managing pull requests
  • Learned how to create and tag releases using semantic versioning
  • Generated automatic release notes
  • Created and merged hotfixes into the release branch and main
